# Toy Tales

A toy list app for managing Andy's toys — view, add, donate, and like toys, backed by a REST API.

## Running the app

Install dependencies:

```
npm install
```

Start the backend (`json-server` on `http://localhost:3001`):

```
npm run server
```

In another tab, start the React app (`http://localhost:5173`):

```
npm run dev
```

## Running tests

```
npm run test
```
